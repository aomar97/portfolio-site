## Principles

- **Prove it, cheaply.** Everything here runs at $0 locally; cloud costs are incurred
  deliberately and briefly, never left idling.
- **Gate the pipeline.** Security and quality checks fail builds, not dashboards.
- **GitOps by default.** The cluster's desired state lives in git.
- **Honest artifacts.** What's live is labeled live; what's code is labeled code.
